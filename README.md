# Calculadora em HTML, CSS e JavaScript
Este projeto é uma calculadora simples desenvolvida com HTML, CSS e JavaScript.
Ela permite realizar operações básicas como soma, subtração, multiplicação e divisão.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Histórico de operações

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
